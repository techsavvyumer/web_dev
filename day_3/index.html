Lecture No# 6

In this video I learned about SEO, Core Web Vitals (CLS, LCP, FID)

CLS:
Commulative Layout Shift means how much page layout changes while loading. Thrive to keep it minimum.
LCP:
Largest Contextual Paint means how much time browser take to render the largest part of page. Should be <=2.5s.
FID:
First Interaction Delay means how much time it takes to start processing the first interaction of user. Should be <=100ms Meta title and description should be present and must reflect the page content for SEO purposes.

Lecture No# 9

In this lecture I learned about the difference between id and class.

ID is a unique identifier to identify individual elements and tags.
ID can also be used in the end of URL with a # sign to go directly to that element when a page loads.
While on the other hand, a class is usually assigned to multiple tags or elements of same group/category.
